# Agentic AI Crash Course

Agentic AI crash course by Codebasics.

## Setup

```bash
uv sync
```

## Usage

```bash
uv run simple-llm-calling/main.py
```

## Dependencies

- langchain
- langchain-google-genai
- langchain-groq
- python-dotenv
- streamlit
- notebook
- pandas
- fpdf2
- pypdf
- langchain-community
- langchain-text-splitters
- pillow
